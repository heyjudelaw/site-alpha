## Rules

1. Use **3-5 tags** per article
2. Always include the **primary company tag** if the story is about a specific company
3. Always include at least one **topic tag**
4. Prefer existing tags over creating new ones
5. If you must create a new tag, add it to this file first
6. Never use version numbers in tags (use `kling` not `kling-30`)
7. Use hyphens for multi-word tags (`ai-safety` not `aisafety`)
